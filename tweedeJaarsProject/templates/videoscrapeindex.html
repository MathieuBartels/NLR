How to use video scrape?

To start using video scrape find a youtube link that contains your object of interest.

For example https://www.youtube.com/watch?v=Gs0RJB30sc4 . This is a video that contains a DJI Phantom 4 quadcopter We insert the youtube link in the form and name the video as we like, (DJIPhantom for example). The video will be stored in /static/video/videoname. The frames will be stored in ../images with a csv that contains the bounding box.

Key Action Description
s Select region of interest The video will start playing, when "s" is pressed you can select the region of interest
r Retrack When the program is following the object, but the bounding box is not optimal, you can retrack it if r is pressed
q Quit Q has 2 actions. It will submit the region of interest after it's drawn, and it will terminate the program when pressed during showing the video
