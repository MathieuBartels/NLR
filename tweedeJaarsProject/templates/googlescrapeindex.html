You are now on the google scraping page. On this page you can see two things; a text input field and an upload field. The text input is meant for the query that the image scraping tool will use to scrape the images. The upload field is meant

Within the text field is a search query posted, for example the name of a drone which is wanted in the database. After that, the upload field is used to upload a golden standard image, which visualize the object from the search query.

The scraped images are compared to the golden standard. The extracted features from the scraped images will be compared to the features of the golden standard. Once the extracted features of the scraped images are similar enough to be classified as a correct image, it will be added to a folder called “Good images”. More about this method can be read in the report.

Some things to be mentioned about a good golden standard is, that it is handy to use a photo which does not contain much noise. While otherwise also features from this noise are used in the comparison.

An image which consists the object as complete as possible is also something to search for. While this gives the opportunity to get many as possible features from the object which can be compared with other images which might have another view on the object.
