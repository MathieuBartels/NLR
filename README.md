## Installation
Install python and blender on your computer 
First clone the repository. 
$ cd NLR-master <br />
$ pip install pipenv <br />
$ pipenv python --python python-3.6.5 <br />
(python 3.6.* works) <br />
(if python is not in your path locate python) <br />
$ pipenv shell <br />
$ pipenv install <br />
$ cd TweedeJaarsProject <br />
